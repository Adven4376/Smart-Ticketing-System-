# Smart Ticketing System

A full-stack ticket booking system built using:
- React (Frontend)
- Spring Boot (Backend)
- MySQL (Database)

👨‍💻 Author: Aditya Josyula  
📅 Year: 2026

This project demonstrates:
- REST APIs
- Concurrency-safe seat booking
- Full-stack deployment (Vercel + Render + Railway)
- React UI with dark mode, login UI, and booking panel

⚠️ This repository is for learning and portfolio purposes.
